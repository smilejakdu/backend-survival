# REST API



## API

* Application Programming Interface
* API는 응용 프로그램에서 사용할 수 있도록, 운영 체제나 프로그래밍 언어가 제공하는 기능을 제어할 수 있게 만든 인터페이스
  * 인터페이스는 컴퓨터 시스템끼리 정보를 교환하는 공유 경계를 의미한다.
